# GH2ME

Unofficial GitHub and Gitea browser client for J2ME platform.

![img](/img/repo_readme.jpg) ![img](/img/source_browser.jpg)

![img](/img/pull_requests.jpg) ![img](/img/issue_comments.jpg)

## Features

- Users, repositories, issues, pull requests, commits
- Search
- Source browser
- Download releases and sources
- Local bookmarks
- Markdown render
- Authorization to access private repositories
- OAuth implementation

## Device support

### Requirements

- MIDP 2.0
- CLDC 1.1
- JSR-75

### Optional requirements

- HTTPS support (recommended)
